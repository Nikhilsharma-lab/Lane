4. What Goes Where

✅ Stays (sidebar)

  • Home — smart focus ordering
  • Inbox — notifications
  • Requests — the ONE work view
  • Ideas — idea board + voting
  • Cycles — time-boxed planning
  • Insights — capacity, impact, pipeline
  • Team — members, health, radar

🗑 Removed / Folded

  • Drafts → tab/filter in Home
  • Stickies → floating widget (already exists)
  • Intake → Requests filtered to predesign
  • Journey View → replaced by grouping in Requests
  • Betting Board → Requests filtered to bet stage (or Cycles)
  • Dev Board → Requests filtered to dev + kanban toggle
  • All Requests → redundant, IS Requests now
  • Initiatives → fold into Cycles or Projects filter
  • Capacity → merged into Insights
  • Impact → merged into Insights
  • Team Health / Radar → merged into Team

5. Pinned Views — The Power User Feature

Any filtered view can be saved and pinned to the sidebar. This replaces dedicated pages for specific workflows.

FOR DESIGNERS

FOR PMs

Users create their own views. Lane can ship default pinned views per role as starter templates.
